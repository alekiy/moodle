Kurs-Schlüssel

Mit dieser Einstellung können Sie den Kurs-Schlüssel manuell einstellen. Dies überschreibt den Kurs-Schlüssel, der vom Moodle-Adminsitrator für diesen Kurs gepflegt wird. Wie auch immer, wenn kein Kurs-Schlüssel für diesen Kurs vorhanden ist, so haben Sie die Möglichkeit, diesen selbst einzutragen und den Kurs so mit einem QUAMP Kurs zu verbinden. Wenn Sie den Kurs-Schlüssel, den der Moodle-Adminstrator pflegt, verwenden wollen, so lassen Sie dieses Feld frei.
